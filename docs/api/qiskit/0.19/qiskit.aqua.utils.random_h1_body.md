# random\_h1\_body

<span id="qiskit.aqua.utils.random_h1_body" />

`random_h1_body(N)`

Generate a random one body integrals.

**Parameters**

**N** (*int*) – the number of spin orbitals.

**Returns**

a 2-D matrix with np.complex data type.

**Return type**

np.ndarray

**Raises**

**ValueError** – invalid number of spin orbitals
